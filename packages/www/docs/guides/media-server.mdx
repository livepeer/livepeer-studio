# Media Server

Livepeer's media server, MistServer, is a full-featured streaming media toolkit.
It enables you to take any media from any location, using any method, and
deliver it seamlessly to anyone, anywhere, in any format.

It's free, open source, and integrates with Livepeer's Streaming Service to
transcode video at scale.

Learn more and download for free at [mistserver.org](https://mistserver.org/).
