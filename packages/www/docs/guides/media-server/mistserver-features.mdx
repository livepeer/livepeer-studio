# MistServer features

MistServer is being optimized to work with Livepeer ‘out of the box’ as much as
possible. At this point in time, that custom transcoding profiles and metric
reporting.

## Custom transcoding profiles

Operators can specify transcoding profiles based on their needs. This allows
flexibility to maximize viewer experience.

## Metrics reporting

MistServer users can look into important transcoding metrics to diagnose issues.
This includes success rate, segment count, ingest endpoint, etc.
